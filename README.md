## **ðŸ”„ Project Workflow**

### **1. Data Collection**

* Designed a **Sales Transaction Form** as a structured data input system
* Implemented **VBA automation** to store transactions into the sales database
* Automated calculation of derived variables, including:

  * Order ID
  * Delivery Time
  * Sales Revenue
  * Total Cost
  * Net Profit

### **2. Data Cleaning**

* Standardized date formats across transaction records
* Removed 33 duplicate records to maintain data integrity
* Handled missing values in Unit Price using LOOKUP from a reference price table
* Generated time-based and financial analysis variables

### **3. Data Analysis**

* Conducted **descriptive statistical analysis** to understand transaction and operational patterns
* Performed **One-Way ANOVA** to test differences in profitability across product categories
* Key finding: profitability differs **significantly** between product categories

### **4. Data Visualization**

* Built an interactive dashboard using Pivot Tables and Pivot Charts
* Core KPIs: Sales Revenue, Total Cost, Net Profit, Total Orders
* Visualized trends, category comparisons, sales channels, and payment methods
* Added slicers to enable dynamic data exploration

---

## **ðŸ“ˆ Key Insights**

* Product profitability differs **significantly** across categories, meaning business performance cannot be evaluated based on sales volume alone
* The **Serum** category delivers the highest profit per transaction, making it a strong candidate for marketing and resource prioritization
* The **Cleanser** category shows the lowest margin, indicating the need for pricing or cost structure optimization
* Digital channels (Official Website and Marketplaces) contribute the majority of total revenue, highlighting the importance of digital-first sales strategies

**Decision Lens:**

These insights enable management to:

* Prioritize high-margin product categories for scaling
* Re-evaluate pricing and cost structures for low-margin categories
* Focus marketing and operational efforts on the most effective sales channels
